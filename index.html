데모 플레이영상

게임 소개

🔥🌊💨 전략과 심리전의 끝! "엘믹스" - 1:1 턴제 카드 배틀 🔥🌊💨

세 개의 지역, 다양한 조합, 그리고 강력한 블록! 상대보다 더 높은 점수를 획득하여 지역을 장악하고 승리를 거머쥐세요!

🎮 게임 방식

  • 각 플레이어는 매 턴 카드를 배치하여 지역을 점령합니다.
  • 지역당 3개의 슬롯: 기본카드 슬롯(하단)과 조합카드 슬롯(상단 2개).
  • 10턴 동안 전략을 펼쳐 최종 점수를 획득하세요!

🔥 속성 시스템

기본 슬롯 카드 효과

  • 바람(💨): 양 사이드에 배치 시, 상대 기본 카드 무효화 (성공 시 +1점).
  • 물(🌊): 상대 기본 속성이 불이면 무효화 (성공 시 +2점).
  • 불(🔥): 해당 지역 점수 +1점.

조합 카드 효과

  • 바람 + 물: 상대 조합 효과 무효화 성공 시 지역점수 +2점, 조합점수 +2점.
  • 물 + 불: 해당 지역 점수 +2점, 조합점수 +2점
  • 불 + 바람: 양옆 지역 점수 +1점 (위치에 따라 최대 2점), 조합점수 +2점

🛑 전략적 차단 시스템: "BLOCK"

각 플레이어는 게임 중 두 번의 BLOCK 기회를 가집니다.

BLOCK을 사용하면 해당 지역의 모든 카드 효과를 무효화합니다.

상대의 핵심 전략을 무너뜨릴 타이밍을 노리세요!

🏆 지역 점령 보너스

5턴 차에서 지역을 장악한 플레이어에게 +2 어드밴티지 부여!

10턴 종료 후 최종 점수 계산!

⏳ 턴 제한

각 턴 45초 제한

9턴 동안 점수 경쟁 후, 10턴에 점수 집계 후 승패 확정!
